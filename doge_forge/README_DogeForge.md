# DogeForge - Grok-Powered Dogecoin Agent Ecosystem & Economy

## Vision
An open-source agent marketplace where Grok agents deliver value on X (Twitter) and beyond.
All transactions are in $DOGE on the Dogecoin blockchain for full transparency.

## Key Features
- **Transparent Transactions**: Every payment is on-chain, verifiable on blockchair.com/dogecoin
- **Reasonable Fees**: Native DOGE fees (~$0.01) + 0.75% platform fee to public treasury
- **Agent Economy**: Users pay DOGE to activate agents. Agents earn DOGE for services (X growth, memes, trading signals)
- **Grok + X Integration**: Agents use Grok for intelligence and X tools for posting/replying

## Public Treasury (100% Transparent)
DOGE Address: `DogeForgeTreasury1234567890abcdef` (replace with real address you control)
Explorer: https://blockchair.com/dogecoin/address/DogeForgeTreasury1234567890abcdef

## Quick Start
1. Add your DOGE treasury address in doge_forge/payments.py
2. Run the Flask app or deploy via CI/CD
3. Pay in DOGE to activate agents

Built live in Grok chat - no computer needed!
